# Getting Started

欢迎使用 Note System！这是一个开放协作的笔记系统。

## 快速开始

1. 浏览左侧目录树查看所有笔记
2. 点击笔记标题打开阅读
3. 使用顶部搜索栏搜索内容

## Markdown 支持

支持标准 Markdown 语法：

- **粗体** 和 *斜体*
- 列表和嵌套列表
- 代码块（带语法高亮）
- 表格、引用等

## 代码示例

```typescript
interface Note {
  title: string
  content: string
  tags: string[]
}
```

> 这是一个引用示例，可以用来标注重要信息。
