# Venezuela Dev Notes

欢迎使用开发笔记系统！这是一个开放协作的笔记集合。

## 快速导航

- [Getting Started](getting-started.md) — 快速上手指南
- [部署指南](guides/deployment.md) — Cloudflare Pages 部署说明

## 如何使用

1. 使用**左侧目录树**浏览所有笔记
2. 点击笔记标题打开阅读
3. 使用**顶部搜索栏**搜索内容

## 支持的 Markdown 语法

- **粗体** 和 *斜体*
- 列表和嵌套列表
- 代码块（带语法高亮）
- 表格、引用等

```typescript
interface Note {
  title: string
  content: string
  tags: string[]
}
```

> 这是一个引用示例，可以用来标注重要信息。
